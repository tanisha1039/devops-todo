# Todo App

A simple full-stack todo application with an Express backend and a vanilla JavaScript frontend.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the server:

```bash
npm start
```

3. Open http://localhost:3000 in your browser.

## Features

- Add todos
- Toggle completion status
- Delete todos
- Simple REST API with in-memory storage

## Notes

- This project is intentionally simple and stores data only in memory.
- Restarting the server resets the todo list.
